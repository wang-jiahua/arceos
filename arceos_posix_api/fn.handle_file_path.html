处理路径并返回规范化后的 FilePath

  • dir_fd - 目录的文件描述符，如果是 AT_FDCWD，则操作当前工作目录

  • path_addr - 路径的地址，如果为 None，则操作由 dir_fd 指定的文件

  • force_dir - 如果为 true，则将路径视为目录

该函数会处理链接并规范化路径
